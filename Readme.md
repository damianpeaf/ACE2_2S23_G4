### Como ejecutar la práctica 2

1. Instalar las dependencias del proyecto

En la carpeta raíz del proyecto `/Practica_2` ejecutar el siguiente comando:

```bash
yarn install
```

2. Añadir páginas a la documentación

Dentro de la carpeta: `/Practica_2/src/content/docs` es posible añadir archivos `.md` para que sean incluidos en la documentación.

Para más información sobre como escribir los archivos `.md` visitar la documentación de [Starlight](https://starlight.astro.build/es/guides/authoring-content/).

### Como ejecutar el proyecto 2

1. Instalar las dependencias del proyecto

En la carpeta raíz del proyecto ejecutar el siguiente comando:

```bash
yarn install
```

2. Ejecutar el `Proyecto 2`

- Para ejecutar la api

```bash
yarn dev:server
```

- Para ejecutar la aplicación movil

```bash
yarn dev:mobile

# En algunos casos, por temas de red, es necesario ejecutar el siguiente comando:

yarn dev:mobile --tunel
```
